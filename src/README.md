# Blog in PHP

See in action: http://blog.ttulka.com

## Project Structure
```
/app
    Application bootstraping and controllers
/assets
    Static resources, CSS, images, etc
/domain
    Domain objects 
/layout
    Layout templates
/mvc
    Simple MVC framework
/view
    View templates
.htaccess
    Webserver settings, environment
index.php
    Entry point
```

## Setup
Do unnecessary changes in `.htaccess`
